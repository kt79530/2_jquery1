Asynchronous

    1. Callback : 콜백은 다름 함수에 인수로 전달되는 함수
    2. Asynchronous : 다름 함수와 병렬로 실행되는 함수를 비동기라고 한다
    3. Promises : 생성코드와 소비코드에 대한 호출이 모두 포함된다
    4. Asynce/Await : async를 함수앞에 붙이면 함수가 promise를 리턴

                    - 현재 효과가 100%완료되면 콜백함수가 실행된다
                    - javascript 명령문은 줄단위[인터프리터 방식]로 실행됨
